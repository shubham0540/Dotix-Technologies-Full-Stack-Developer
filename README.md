# Dotix — Full Stack Project

A simple full-stack project with a Next.js frontend and Express.js backend for managing jobs.

## Project structure

- `backend/` — Express.js API and services. (Now tracked as a normal directory)
- `frontend/` — Next.js frontend application

## Quick start

Prerequisites:
- Node.js 18+ and npm

Run backend:
```bash
cd backend
npm install
npm start
```

Run frontend:
```bash
cd frontend
npm install
npm run dev
```

## Notes
- `node_modules/` are ignored by `.gitignore`.
- The original backend submodule metadata was preserved in `_backend_git_backup/`.
